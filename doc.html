Instalare CLI

Cerințe

  • Windows 10, Linux, Mac OS X
  • Node.js versiunea 14.15.3+ sau
  • PHP 8.0.0+ sau
  • Java SE 14+ sau
  • Ruby 2.7.2+ sau
  • Python 3.9.1+

Instalare

Homebrew

brew install solaine-cli

NPM

npm install solaine-cli

YARN

yarn install solaine-cli

PIP

pip install solaine-cli

Anaconda

conda install solaine-cli
